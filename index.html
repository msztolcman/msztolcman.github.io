Toys

  • Sendria (formerly MailTrap) is a SMTP server designed to run in your dev/test environment, that is designed to catch any email you or your application is sending, and display it in a web interface instead of sending to real world.
  • SMTPc is simple SMTP client for easy mail sending using CLI. It's dedicated for developers, however it's easy to use and every CLI user will be satisfied using this.
  • subst - simple utility to replace one string into another in given list of files.
  • versionner - helps manipulating version strings in the project.
  • ff - Easily search and process files.
  • integresql-client-python - Python client for IntegreSQL
  • irename allows to easy rename many files using Your favorite editor.
  • hashfile - calculate hash or checksum in many formats.
  • highlight - Highlight the found text.
